Windows好用plugin-powertoys

發表於 2020-09-05

在OS X中，常常會看到使用者會透過鍵盤開啟一個放大鏡，直接打上路徑或是軟體名，就可以直接開啟(找張圖)

現在，Windows也在最近提供一個擴充功能(powertoys)，讓大家也能直接使用這個功能

他除了之前提到的spotlight功能，他還提供了其它很實用的功能

Color Picker

可以直接透過組合鍵取得畫面元素的色碼，並放進剪貼簿中

FancyZones

可以將軟體的布局自定義成你想要的位置，如果你有超寬螢幕時超級推薦!!!

File explorer Preview

可以讓你在檔案總管中除了原本可以預覽的圖片或是影片之外
在安裝完後，新增了

  1. svg (小圖示跟預覽畫面都可以)
  2. markdown預覽
    從此，我再也不用透過web來預覽svg了，太棒了!!!

Image Resizer

可以透過滑鼠右鍵的選單直接將圖片縮小或放大

Keyboard Manager

可以將你的鍵盤從新remap， 而且還支援個別程式自定義
如果不是重度需求者，基本上可以取代autohotkey了

PowerRename

在Windows上，如果要大量重新命名檔案，通常只能透過提示命令字元下指令完成
但在這個工具上，他提供你圖形化介面修改，另外，還可以透過正規表達式來進行修改的動作，功能十分強大!!

Powertoys Run(重點)

在任何程式開啟的狀態，按下alt+space，就可以直接開啟搜尋工具，打字即可開啟需要的服務
目前支援的指令有

  1. 網址
  2. 應用程式
  3. 資料夾/檔案
  4. cmd直接運行(“>ipconfig” 就可以執行ipconfig)
  5. 開啟已開啟的應用程式

網址，應用程式這兩個比較單純，打上你要的，看到你的目標，enter案下去就對了

資料夾/檔案這點要補充一下
當你找到你要開啟的資料夾時，如果按下ctrl+shift+c，就可以直接透過cmd直接開啟這個資料夾，在需要下指令時非常好用

開啟已開啟的應用成，因為它的優先順序問題，如果在搜尋頁面打上應用程式名，已經開啟的選項會放在選單的最下方，直接按下方向鍵上再選擇即可

Shortcut Guide

如果忘記快捷鍵，除了可以打開設定之外，還可以透過長按win key，他會將目前有的快捷鍵都顯示在螢幕中，大家可以自己玩玩看喔!!

Code Editor介紹(上)

發表於 2020-09-15

工欲善其事，必先利其器，要學習開發網頁的第一步，就是要先有個用的順手的編輯器，而目前比較常見的就下面列的這幾種

  1. Sublime
  2. VSCode
  3. Atom
  4. JetBrain系列
  5. Vim
  6. Emcas
    …
    上面這些是我比較常看到的，但因為今天主要是就我有使用過的部分來做比較，因此今天主要會以介紹VSCode為主

先介紹一下今天的主角–VSCode

介紹部分就交給維基百科吧，這邊就不多作介紹了
身為一個主流的Editor，基本上的上色功能是一定有的

這邊想介紹我比較常用的Plugin，讓在打code的時候可以越打越開心!!
首先
主題部份我是使用One dark pro
但主題部分我覺得找一個看得順眼的最重要，在vscode中有很多的主題供大家選擇

接下來強烈推薦先安裝以下這個套件
Sublime Text Keymap and Settings Importer
這個可以幫你把原本在sublime中的設定檔以及sublime的熱鍵複製過來，很好用的!!

下面這些是我目前比較常使用到的套件

  • Advanced-new-file
    • 可以透過鍵盤直接新增檔案，手就可以不用離開鍵盤了
  • Auto Rename Tag
    • 修改tag時修改開頭的tag，尾巴的tag也會一起修正
  • GitLens — Git supercharged
    • 強大的git套件，還可以透過圖形化介面來修改內部的設定檔
  • Guides
    • 會幫你的縮排畫個線，讓你比較好對其
  • Prettier - Code formatter
    • 幫你自動排版的好用工具，必裝!!
  • WakaTime
    • 免費計時你打code的歷程以及時間，還可以結合git紀錄，了解每個commit花了多久時間
  • Vetur
    • 開發vue必裝

VSCode-2

發表於 2020-09-16

接下來的內容是以有安裝過Sublime Text Keymap and Settings Importer當作前提來做說明的!!

已經幫vscode安裝好套件後，那接下來就讓我們來看一下好用的熱鍵吧

  • Ctrl+d => 可以幫你自動選取下一個相同的單字，並可以一起修改

  • 按下滑鼠中鍵 => 可以垂直選取內容

  • alt+左右方向鍵 => 可讓游標一次跳一個單字

  • home/end => 將游標移動到整行的頭/尾端
    這三個技巧可以合併使用

    在這裡我把Reducer打成Reduce，並忘記加上分號
    可以按住滑鼠中鍵選取你要的範圍，透過alt跳到這個單字最後，加上r之後按下end，加上;就可以了
    當然也可以用上面提到的ctrl+d將reduce選起來，把它改成reduce，在按下end加上;也行

  • ctrl+shift+d => 可以一次複製貼上一行

  • alt+ 方向鍵上/下 => 將所在的那行上下移動

  • 在不反白任何字的情況按下ctrl+c/x => 可以複製一行/剪下一行

    我想要在google登入的按鈕下增line跟facebook的登入按鈕
    就可以透過ctrl+shift+d 複製貼上兩行，把字改掉
    如果希望facebook跟line交換位置，就按下alt+上/下交換位置就可以了

  • ctrl+p => 可透過文字搜尋檔案，並開啟他(預設會選取上一次開啟的檔案)

    這個超好用的，有時候需要到其他檔案參考裡面的內容，想要回去原本的檔案做修改，就按下ctrl+p直接按下enter就可以了
    也可以透過打字把需要的檔案找出來，這個搜尋功能有提供模糊搜尋，只要打出幾個字就會找到你要的檔案了

其他還有很多的快捷鍵等大家來發掘，上面介紹的是我在開發的時候用的最頻繁的幾組快捷鍵，當時知道有這些功能的時候可是超級興奮的，所以在這邊分享給大家，也歡迎大家可以一起交流更方便的快捷鍵瞜~~

那我們就明天見!!
